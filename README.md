# 🌾 FarmGuard
> AI-powered animal intrusion detection system for farmers

## What it does
Detects animals near farm fields using a live camera feed.
Sends an instant Telegram alert with a snapshot when an animal is detected.
Built to solve real crop damage faced by Indian farmers during harvest season.

## Tech Stack
- Python
- YOLOv8 (Ultralytics)
- OpenCV
- Telegram Bot API
